Welcome to a regional COVID-19 virus tracker. Our goal is to provide easy to read, easy to access statistics on COVID-19. The process is simple and easy, so after reading this, navigate to the "Tracker" page when you're ready. The map is interactive and should already be focused on Canada so you can begin searching for your region right away. After clicking on any selectable region, important statistics regarding the impact of COVID-19 there are shown. From there it is as simple as reading through the list and finding what you need. We told you it was simple.

Project Overview

The Covid-19 Tracker for Canada is designed to provide real-time updates on COVID-19 statistics across different regions of Canada. The main objective of this project is to keep Canadians informed and aware of the current pandemic situation in their respective regions.

Features

  • Real-time COVID-19 statistics
  • Region-specific data
  • Interactive map with clickable regions
  • Light and dark theme toggle

Data Sources

The data displayed on this tracker is sourced from reliable APIs, including the Covid19Tracker.ca API. We strive to ensure the accuracy and timeliness of the data.

Technologies Used

This project is built using the following technologies:

  • Supabase
  • Leaflet.js for interactive maps
  • JavaScript
  • HTML & CSS
